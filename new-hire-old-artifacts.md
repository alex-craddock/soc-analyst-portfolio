# New Hire Old Artifacts – Investigation

Platform: TryHackMe  
Category: Digital Forensics / Artifact Analysis  

## Scenario

In this lab we investigate historical artifacts on a system to determine suspicious activity.

## Investigation Steps

### Step 1 – Initial Review

We begin by reviewing system artifacts to identify potential indicators of suspicious behaviour.

### Step 2 – Artifact Analysis

Artifacts examined:

- File system artifacts
- User activity traces
- Historical files

### Step 3 – Suspicious Findings

During the investigation several suspicious indicators were discovered including unusual files and evidence of previous user activity.

## Indicators of Compromise (IOCs)

- Suspicious files
- User activity artifacts
- Unusual file locations

## Lessons Learned

This investigation demonstrated how historical artifacts can reveal past activity on a system and help analysts identify potential incidents.
